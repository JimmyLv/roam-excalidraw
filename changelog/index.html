Change log

March 14th, 2021

  • MVP release

March 18th, 2021

  • Added further settings
    • full-screen-width is the gap in percentage of window width and window height when in editing mode. 0 will make Excalidraw completely cover the hosting Roam page.
  • Added support for easier styling 
    • :root {
    --bg-ex-dark: 	#121212;
    --cl-ex-dark: 	lightgray;
    --bg-ex-light:	white;
    --cl-ex-light:	#121212;
  }
  • You can now double click on the image embedded on the Roam page to access the underlying block for editing.
  • Fixed sticky Zen Mode and Grid Mode.
  • Other Minor stability improvements.
